* create a empty folder (myproject) and do the following

```sh
cd myproject
npm -v // 5.3.0
npm init
npm install react react-dom react-router react-scripts express --save

mkdir src
mkdir public
touch src/index.js
touch public/index.html

```

* open `package.json`
* put an entry under scripts
* `    "start": "react-scripts start" `
